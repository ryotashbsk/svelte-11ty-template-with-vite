# 11ty + Svelte + Vite

static site generator: [11ty](https://www.11ty.dev/)  
frontend framework: [Svelte](https://svelte.dev/)  
build: [Vite](https://vitejs.dev/)

## install

```
yarn
```

## development

```
yarn dev
```

## production

```
yarn prod
```

## preview

production と同じものを[http:localhost:5000](http:localhost:5000)で閲覧可  
リリース時の最終確認用として使える

```
yarn preview
```

## format

prettier でフォーマット

```
yarn format
```
